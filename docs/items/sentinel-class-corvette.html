Sentinel Class Corvette

vehicle

Overview

A corvette build by Steiger Aerospace, these are used by the military as scout ships, and even a few are purchased by the Marshals to help interdicting pirates. Like most Steiger builds they are rugged and highly customizable. While most of this class are space only ships, there are a view variants that are able to land on low to normal atmospheric planets.
